Basic Types

TypeScript provides static types to catch errors early. Here's how to use basic types like string, number, boolean, arrays, and tuples.


// Basic Types
let name: string = "Alice";
let age: number = 30;
let isStudent: boolean = true;
let hobbies: string[] = ["reading", "coding"];
let tuple: [string, number] = ["Bob", 25];
let anyValue: any = "Can be anything";
let unknownValue: unknown = "Needs type checking";

displayOutput("basic-types-output", `Name: ${name}, Age: ${age}, IsStudent: ${isStudent}`);
displayOutput("basic-types-output", `Hobbies: ${hobbies.join(", ")}`);
displayOutput("basic-types-output", `Tuple: ${tuple[0]}, ${tuple[1]}`);
displayOutput("basic-types-output", `Any: ${anyValue}, Unknown: ${typeof unknownValue === 'string' ? unknownValue : 'Not a string'}`);

Interfaces

Interfaces define the shape of objects, ensuring consistent structure and behavior.


interface Person {
  name: string;
  age: number;
  greet(): string;
}

const person: Person = {
  name: "Charlie",
  age: 28,
  greet() {
    return `Hello, I'm ${this.name}!`;
  }
};

displayOutput("interfaces-output", `Interface: ${person.greet()}`);

Classes and Inheritance

Classes provide object-oriented programming with inheritance for reusable code.


class Animal {
  constructor(public name: string) {}
  move(distance: number): string {
    return `${this.name} moved ${distance}m.`;
  }
}

class Dog extends Animal {
  bark(): string {
    return `${this.name} says Woof!`;
  }
}

const dog = new Dog("Rex");
displayOutput("classes-output", `Class: ${dog.move(10)}`);
displayOutput("classes-output", `Inheritance: ${dog.bark()}`);

Generics

Generics allow reusable functions that work with any type while maintaining type safety.


function demonstrateGenerics(items: T[], prefix: string): string {
  return `${prefix}: ${items.join(", ")}`;
}

const numbers = [1, 2, 3];
const strings = ["apple", "banana", "cherry"];
displayOutput("generics-output", `Generics with numbers: ${demonstrateGenerics(numbers, "Numbers")}`);
displayOutput("generics-output", `Generics with strings: ${demonstrateGenerics(strings, "Fruits")}`);

Enums

Enums define a set of named constants, useful for fixed sets of values.


enum Direction {
  Up = "UP",
  Down = "DOWN",
  Left = "LEFT",
  Right = "RIGHT"
}

const direction: Direction = Direction.Right;
displayOutput("enums-output", `Enum: Current direction is ${direction}`);

Union and Intersection Types

Union types allow multiple types; intersection types combine multiple interfaces.


type ID = string | number;
interface Business {
  company: string;
}
interface Employee {
  role: string;
}
type Worker = Business & Employee;

const id: ID = "EMP123";
const worker: Worker = { company: "xAI", role: "Developer" };
displayOutput("union-intersection-output", `Union Type: ID=${id}`);
displayOutput("union-intersection-output", `Intersection Type: Worker at ${worker.company} as ${worker.role}`);

Type Assertion

Type assertions let you tell TypeScript the type of a value when it's ambiguous.


let someValue: any = "This is a string";
let strLength: number = (someValue as string).length;
displayOutput("type-assertion-output", `Type Assertion: String length is ${strLength}`);
